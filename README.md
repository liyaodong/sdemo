# sdemo

参考：https://github.com/foru17/StaticPage

用处：只要保留`bin/sdemo.sh`文件copy出来。在任意位置执行`sh sdemo.sh`

## 目录结构

```
.
├── Gemfile              # sass和coffee支持
├── Gruntfile.coffee
├── README.md
├── assets               # 编译后的文件
│   ├── css
│   └── js
├── .bowerrc             # Bower配置
├── bin
│   └── sdemo.sh         # 就是脚本咯...
├── bower.json
├── index.html
├── package.json
└── src                  # 代码写在这
```
